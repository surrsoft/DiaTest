Строка запроса

Примеры запросов

  • поиск по кадастровыму номеру
    77:1:1014:12
  • поиск по диапазону кадастровых номеров
    77:1:1014:0-1000
  • поиск всех объектов кадастрового квартала по номеру кадастрового квартала
    77:1:1014:*
  • используя точку с запятой ";" можно строить запросы любой длины
    77:1:1014:12;77:1:1014:0-1000;77:1:1014:*
  • кадастровые номера можно записывать как с лидирующими нулями (например 77:01:0001014:0012) так и без них (например 77:1:1014:12)
